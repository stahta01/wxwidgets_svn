\section{\class{wxXmlAttribute}}\label{wxxmlattribute}

Represents a node attribute.

Example: in {\tt <img src="hello.gif" id="3"/>}, {\tt ``src"} is attribute with value
{\tt ``hello.gif"} and {\tt ``id"} is a attribute with value {\tt ``3"}.
