\section{\class{wxPropertyAccessor}}\label{wxpropertyaccessor}

This class allows the user to set and retrieve a \helpref{wxPropertyInfo}{wxpropertyinfo} 
declared in a \helpref{wxClassInfo}{wxclassinfo} object.

The main functions are \helpref{SetProperty}{wxpropertyaccessorsetproperty},
\helpref{GetProperty}{wxpropertyaccessorgetproperty} (for setting/getting a normal property)
and \helpref{AddToPropertyCollection}{wxpropertyaccessoraddtopropertycollection},
\helpref{GetPropertyCollection}{wxpropertyaccessorgetpropertycollection} (for adding elements
or retrieving them from "collections" - i.e. arrays).
